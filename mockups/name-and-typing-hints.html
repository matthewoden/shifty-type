Get your name, then guide the typing

Four changes to smooth onboarding and make the board teach itself. The phones below are the real screens; only the dashed-green pieces are new.

▢ dashed green = the new bit
1

“Get started” asks your name in a card

Today, tapping Get started with no saved name slides an input in above the button — so you have to reach back up and tap it again. Instead, one tap brings up a name card. Type it, tap Let’s go, and you’re in the match. Saved once, never asked again.

Why

The old flow makes you tap the same button twice with a context switch in between — it reads as “nothing happened.” A card is an unmistakable “answer this, then continue.”

Same field is reused in the settings view, so the rename pattern is learned once.

2

A quiet settings view

Instead of a loud chip on Home, a small gear in the corner opens a settings screen. Right now it holds one thing that matters — your name — with room to grow (bell, install, restore games) later. Renaming saves instantly.

Why

A settings view keeps Home focused on playing. Ring me for every game is the master switch — flip it on and every current and future match nudges you, no per-game tapping.

The per-match bell stays for one-off control; it just reads “on” once the master switch is set.

The gear is the only new Home pixel; everything else there is untouched.

3

The opening turn shows a cursor

On your very first turn the board is empty — nothing tells you it’s time to type, or where the tiles will land. A blinking cursor sits at the first tile’s spot, and the hint invites the tap. The keys are all live, because an opener can be any word.

Why

An empty board with a hidden custom keyboard is a dead end for a first-timer. The cursor borrows the same pulsing caret the draft already uses, so it reads instantly as “type here.”

It also previews where tiles appear, so the first letter doesn’t jump in from nowhere.

4

Guided keys after the first word

From the second word on, your word must start with the tail of the last one. The deck now lights the letters you can start with and greys out the rest — the tutorial’s coloring, everywhere. As soon as your overlap is locked in, every key wakes back up and you type freely. (The engine already ignores illegal keys — this just makes it visible.)
